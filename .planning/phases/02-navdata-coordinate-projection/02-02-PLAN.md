## Trust Boundaries

|----------|-------------|
| Hand-authored navdata → sim core | Real-world coordinates/headings are transcribed by hand into eggw.py; a transcription typo silently corrupts "real" data |
| sim core (navdata) → render | render/radar.py reads navdata + projection output only; it must never mutate navdata (frozen models enforce this) |
